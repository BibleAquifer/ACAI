# Key
**Key**. 
An object used to lock or unlock doors and gates (acai). 


**Key References for Key**: 
MAT 16:19, REV 1:18, REV 20:1. 




In the New Testament (BSB), words representing “Key” are translated as 
*keys* (1x), *key* (2x).
