# Drachma
**Drachma**. 
A Greek silver coin with about the same value as a Roman denarius (acai). 


**Key References for Drachma**: 
LUK 15:8, LUK 15:9. 




In the New Testament (BSB), words representing “Drachma” are translated as 
*silver coins* (1x), *of them* (1x), *coin* (1x).
